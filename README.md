# Fashion-Match

A comprehensive wardrobe management platform that helps users maximize their existing wardrobe and make informed fashion decisions through intelligent outfit recommendations and analytics.

## Overview

Fashion-Match is designed for individuals who want to dress well without the complexity of styling themselves. The platform allows users to upload their clothing items, generates outfit combinations based on their wardrobe, identifies missing pieces in suggested outfits, and provides direct purchase links for these items from online stores.

## Features

- **Wardrobe Management**: Upload and organize clothing items by category (tops, bottoms, shoes, outerwear, accessories)
- **Smart Outfit Recommendations**: Automatically generates outfit combinations based on uploaded items
- **Missing Item Detection**: Identifies missing pieces in suggested outfits and provides purchase links
- **Weather-Based Suggestions**: Offers outfit recommendations based on current weather conditions
- **Wardrobe Analytics**: Provides insights on wardrobe value, color/style distribution, and unused items
- **Smart Combinations**: Generates combinations based on occasion and weather with color harmony analysis

## Technology Stack

- **Frontend**: React
- **Backend**: Node.js, Express.js
- **Database**: SQLite
- **APIs**: Weather API integration

## Setup

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/SimgeBozdogan/Fashion-Match.git
cd Fashion-Match
```

2. Install frontend dependencies:
```bash
npm install
```

3. Install backend dependencies:
```bash
cd server
npm install
cd ..
```

4. Start the backend server:
```bash
cd server
npm start
```

5. Start the frontend (in a new terminal):
```bash
npm start
```

The application will be available at `http://localhost:3000`.
